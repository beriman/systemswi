1 Tujuan setup ini

Kita mau sistem SWI bisa mengambil data Instagram @fragrantions secara resmi lewat Instagram Graph API, bukan scrape manual.

  • Ambil daftar post dan reels
  • Ambil caption dan tanggal posting
  • Ambil permalink Instagram
  • Ambil media URL / thumbnail
  • Simpan dokumentasi event ke systemswi
  • Tampilkan sebagai portfolio Fragrantions
Penting: Jangan pakai bot/scraping paksa. Instagram sering memblokir akses server dan datanya tidak stabil. API resmi lebih aman untuk data bisnis.

2 Checklist paling cepat

Kalau mau versi super singkat, Wapiq cukup ikuti ini:

  1. Pastikan @fragrantions adalah akun Business atau Creator.
  2. Hubungkan @fragrantions ke Facebook Page.
  3. Buat Meta Developer App.
  4. Tambahkan produk Instagram API / Facebook Login for Business.
  5. Generate token dengan permission: instagram_basic dan pages_show_list.
  6. Jalankan endpoint: /me/accounts?fields=name,id,instagram_business_account.
  7. Ambil Instagram Business Account ID.
  8. Test endpoint media Instagram.
  9. Buat long-lived access token.
  10. Kirim data ke Beriman/HemuHemu untuk dipasang ke systemswi.

3 Pastikan @fragrantions Business / Creator

Di aplikasi Instagram, login ke akun @fragrantions, lalu cek tipe akun.

  1. Buka Instagram @fragrantions.
  2. Masuk ke Settings and activity.
  3. Pilih Account type and tools.
  4. Pastikan akun sudah Business Account atau Creator Account.
Rekomendasi: Untuk Fragrantions, gunakan Business Account agar lebih cocok untuk integrasi event dan brand.

4 Hubungkan ke Facebook Page

Instagram Graph API biasanya membutuhkan akun Instagram Business/Creator yang tersambung dengan Facebook Page.

  1. Buka Instagram @fragrantions.
  2. Masuk ke Settings and activity.
  3. Buka Business tools and controls.
  4. Pilih Connect or create Facebook Page.
  5. Pilih Page yang sudah ada, atau buat Page baru bernama Fragrantions.
Pastikan: Akun Facebook Wapiq / admin punya akses penuh ke Page Fragrantions di Meta Business Suite.

5 Buat Meta Developer App

Buka Meta for Developers:

https://developers.facebook.com/
  1. Login memakai akun Facebook yang punya akses ke Page Fragrantions.
  2. Klik My Apps.
  3. Klik Create App.
  4. Pilih tipe app yang berhubungan dengan Business / Business Integration.
  5. Nama app yang disarankan:
SWI Fragrantions Instagram Sync

Setelah app dibuat, masuk ke dashboard app tersebut.

6 Tambahkan produk & permission

Di dashboard Meta App, buka Add products, lalu tambahkan produk yang berkaitan dengan Instagram API, biasanya salah satu dari:

Instagram Instagram API with Facebook Login Facebook Login for Business

Permission minimal

instagram_basic
pages_show_list

Fungsi permission:

  • instagram_basic: membaca profil dan media Instagram Business/Creator.
  • pages_show_list: melihat daftar Facebook Page yang terhubung ke akun Facebook.

Permission tambahan kalau nanti dibutuhkan

instagram_manage_insights
instagram_manage_comments
instagram_content_publish
Tahap awal cukup: instagram_basic dan pages_show_list. Jangan centang terlalu banyak permission dulu.

7 Generate token di Graph API Explorer

Buka Graph API Explorer:

https://developers.facebook.com/tools/explorer/
  1. Di kanan atas, pilih app: SWI Fragrantions Instagram Sync.
  2. Klik Generate Access Token.
  3. Centang permission:
instagram_basic
pages_show_list
  1. Login/authorize menggunakan akun Facebook yang punya akses ke Facebook Page Fragrantions.
  2. Simpan sementara token yang keluar. Ini biasanya masih short-lived token.
Rahasia: Access token jangan di-share ke grup umum. Token ini bisa memberi akses baca data akun Instagram/Page.

8 Ambil Instagram Business Account ID

Di Graph API Explorer, jalankan endpoint ini:

GET /me/accounts?fields=name,id,instagram_business_account

Kalau sukses, hasilnya kira-kira seperti ini:

{
  "data": [
    {
      "name": "Fragrantions",
      "id": "FACEBOOK_PAGE_ID",
      "instagram_business_account": {
        "id": "INSTAGRAM_BUSINESS_ACCOUNT_ID"
      }
    }
  ]
}

Catat dua hal penting:

Facebook Page ID:
...

Instagram Business Account ID:
...

9 Test ambil media Instagram

Setelah mendapatkan Instagram Business Account ID, jalankan:

GET /INSTAGRAM_BUSINESS_ACCOUNT_ID/media?fields=id,caption,media_type,media_url,permalink,timestamp,thumbnail_url

Contoh:

GET /17841400000000000/media?fields=id,caption,media_type,media_url,permalink,timestamp,thumbnail_url

Hasil sukses kira-kira:

{
  "data": [
    {
      "id": "18000000000000000",
      "caption": "Road to Fragrantions...",
      "media_type": "VIDEO",
      "media_url": "https://...",
      "permalink": "https://www.instagram.com/reel/...",
      "timestamp": "2026-04-..."
    }
  ]
}
Catatan: Untuk carousel, nanti biasanya butuh request tambahan ke children media. Untuk tahap awal, endpoint di atas sudah cukup untuk validasi koneksi.

10 Buat Long-lived Access Token

Token dari Graph API Explorer biasanya pendek masa aktifnya. Untuk systemswi, kita butuh token yang lebih panjang, biasanya sekitar 60 hari.

Gunakan endpoint berikut:

https://graph.facebook.com/v20.0/oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=SHORT_LIVED_TOKEN

Ganti:

  • APP_ID: App ID dari Meta Developer App.
  • APP_SECRET: App Secret dari Meta Developer App.
  • SHORT_LIVED_TOKEN: token yang didapat dari Graph API Explorer.

Hasilnya kira-kira:

{
  "access_token": "LONG_LIVED_ACCESS_TOKEN",
  "token_type": "bearer",
  "expires_in": 5183944
}
Simpan baik-baik: Catat token dan tanggal expiry. Jangan masukkan token ke file publik, screenshot publik, atau Google Doc terbuka.

12 Troubleshooting umum

Akun Instagram tidak muncul

  • @fragrantions belum Business/Creator.
  • @fragrantions belum connect ke Facebook Page.
  • Akun Facebook yang dipakai tidak punya akses ke Page.
  • Page belum benar-benar linked ke Instagram di Meta Business Suite.

Error permission

Pastikan token punya permission:

instagram_basic
pages_show_list

Error unsupported get request

Biasanya ID salah atau token tidak punya akses. Ulangi endpoint:

/me/accounts?fields=name,id,instagram_business_account

Media URL tidak permanen

media_url bisa berubah/expire. Untuk dokumentasi jangka panjang, simpan minimal:

permalink
caption
timestamp
media_type
thumbnail_url
