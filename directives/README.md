# Directives (SOP)

Folder ini berisi **Standard Operating Procedures** yang mendefinisikan *what to do* untuk setiap task.

## Format Directive

```markdown
# [Nama Directive]

## Tujuan
Jelaskan apa yang ingin dicapai.

## Input
- Input yang diperlukan

## Tools/Scripts
- Script atau tools yang akan digunakan

## Output
- Output yang diharapkan

## Steps
1. Langkah pertama
2. Langkah kedua
...

## Edge Cases
- Kasus khusus yang perlu diperhatikan
```

## Best Practices

✅ **DO**:
- Tulis dalam bahasa yang jelas dan terstruktur
- Sertakan contoh konkret
- Update directive setelah belajar dari error

❌ **DON'T**:
- Jangan terlalu abstrak
- Jangan skip edge cases
- Jangan lupa update setelah ada perubahan
