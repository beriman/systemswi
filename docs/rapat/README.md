| File | Deskripsi |
|------|-----------|
| `risalah-rups-2026-01-11.md` | Risalah RUPS tanggal 11 Januari 2026 |

## Cara Update

1. Untuk risalah rapat baru, buat file dengan format: `risalah-[jenis]-YYYY-MM-DD.md`
2. Update task tracking sesuai progress secara berkala
3. Gunakan checkbox `[x]` untuk menandai task yang selesai
